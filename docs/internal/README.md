# Internal Notes

This directory stores historical planning notes.

这些文件用于维护者上下文，不属于公开使用面的主要入口（如 `README.md`、`SECURITY.md`、examples 或运行时代码）。

Recommended entry points:
- `CLAWeb-STAGE-SUMMARY.md`: current stage baseline / 当前阶段基线
- `CLAWeb-PLAN.md`: archived planning draft / 历史规划草稿
- `CLAWeb-RELEASE-NOTES-DRAFT-2026-03-13.md`: release note draft / 发布说明草稿
